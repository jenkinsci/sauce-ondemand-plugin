The selected browser information will be available through the following environment variables:
  • SELENIUM_PLATFORM - The operating system of the selected browser
  • SELENIUM_VERSION - The version number of the selected browser
  • SELENIUM_BROWSER - The browser name of the selected browser.
  • SELENIUM_DRIVER - Contains the operating system, version and browser name of the selected browser, in a format designed for use by the Selenium Client Factory library
  • SELENIUM_DEVICE - The device of the selected browser
  • SELENIUM_DEVICE_ORIENTATION - The orientation for the device of the selected browser
  • SAUCE_ONDEMAND_BROWSERS - A JSON-formatted string representing the selected browsers
